# Awesome API

Projeto desenvolvido em Python com objetivo de capturar a cotação de diferentes moedas.

## Dicionario de dados

| Key | Label |
| --- | ----- |
| bid | Compra |
| ask | Venda |
| varBid | Variação |
| pctChange | Porcentagem Variação |
| high | Máximo |
| low | Mínimo |
